HOW TO USE?
Ensure vulner.sh, back.rb, svc2exp and excl_ips  are in the same directory.

File description
vulner.sh is main bash script for vulner project.
back.rb is used by vulner.sh to set msfconsole session to background.
svc2exp is used by vulner.sh to map vulnerable service to exploit module.
excl_ips is a text file for IP address exclusion via manual entry.
